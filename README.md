# Test Plugin for the thonny IDE

## Installation

```bash
pip install -r requirements.txt --user
```

## Start the plugin with thonny

```bash
cd /path/to/thonny/
PYTHONPATH=/path/to/thonny-testplugin/ python -m thonny
```
